## 홍성남
 
뒤늦게 찾은 흥미에 확신을 가졌고, 확신을 현실로 만들어 즐겁게 개발하며 성장하고 싶습니다.

### [개인 프로젝트 - Gitmap](https://github.com/Seongnam-si/git-map)
로컬 커밋 시점의 위치 정보를 수집해 개발자의 활동 지역을 Github 프로필에 시각화해주는 CLI 기반 도구입니다.   

### [개인 프로젝트 - 숫자야구](https://github.com/Seongnam-si/rn-baseball)
서로 다른 숫자를 추리, 정답을 맞추는 간단한 캐주얼 게임입니다.  
게임 결과와 기록을 함께 보여주며, 짧은 시간 몰입 가능한 구조로 만들었습니다.  
현재 앱스토어에서 다운로드 가능합니다.

### [팀 프로젝트](https://github.com/Seongnam-si/The-Chuck)
척척이라는 장난감을 자유롭게 만들어 볼 수 있는 3D 시뮬레이션 서비스입니다.  
어린 시절 문구점 앞 뽑기 기계를 통해 쉽게 만나볼 수 있는 장난감이었지만,  
이제는 찾기 힘든 척척이를 어른이들에게 제공하여 즐거운 시간을 선사합니다.
